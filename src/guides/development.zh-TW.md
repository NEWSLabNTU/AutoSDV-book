# 開發指南

本節為想要擴充、修改或貢獻至 AutoSDV 平台的開發者提供資訊。本指南涵蓋五個關鍵領域：

1. **原始碼說明** - 儲存庫結構與組織
2. **版本控制** - 使用 Git 超專案與子模組
3. **感測器組件與驅動程式** - 可用的感測器及其配置
4. **感測器套件配置** - 感測器整合與校正
5. **車輛介面** - 控制系統與車輛參數
